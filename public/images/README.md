
# Imagens e Vídeos - ASANTOS DESPORTO

Todos os arquivos listados abaixo devem estar localizados em `/public/images/` para garantir o funcionamento do Guia Visual. Use sempre caminhos absolutos no código (ex: `/images/arquivo.mp4`).

### Lista de Arquivos Obrigatórios:
- `Perfil.jpg`: Foto de perfil profissional.
- `SupinoReto.mp4`: Demonstração de Supino Reto (Barra/Halteres).
- `SupinoInclinado.mp4`: Demonstração de Supino Inclinado.
- `PeckDeck.mp4`: Demonstração de Voador / Peck Deck.
- `RemadaCurvada.mp4`: Demonstração de Remada Curvada.
- `exercise-close-grip-bench-press.mp4`: Demonstração técnica de Supino Fechado.
- `PuxadaAlta.gif`: Demonstração de Pulley / Puxada Alta.
- `Agachamento.gif`: Demonstração de Agachamento Livre.
- `LegPress.gif`: Demonstração de Leg Press 45º.
- `Desenvolvimento.gif`: Demonstração de Desenvolvimento de Ombros.

### Notas:
1. Os ficheiros MP4 são preferíveis para demonstrações complexas por permitirem maior nitidez e controlo de reprodução.
2. Certifique-se de que os nomes dos ficheiros respeitam a capitalização exata (ex: `SupinoReto.mp4` vs `supinoreto.mp4`).
